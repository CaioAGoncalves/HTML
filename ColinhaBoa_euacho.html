Diferença entre linguagem interpretada e compilada Pequeno dicionário de comandos HTML
  • [!DOCTYPE html] -- Essa linha e a próxima são obrigatórias --

  • [html lang ="pt-br"] -- Todo o comando fica dentro deste html (como se fosse o main)/Pt-br é auto-explicativo

  • [head] -- providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos

  • [title>Wikipédia - Caio/title] -- Nome do site no topo da página --

  • [meta charset="utf-8"] --Descrevem o conteúdo do seu site para os buscadores/ UTF-8 UTF-8 é a página de código universal para internacionalização usado amplamente é o padrão

  • [br] pula uma linha // [p] pula pro próx parágrafo // [b] negrito // [i] itálico

  • [ul] uma lista desordenada // [ol] uma lista ordenada // [li] representar um item que faz parte de uma lista

  • [img] coloca uma imagem Exemplo de como usar: (img alt= "Apenas Pinguim" src= Pinguu.jpg height="450px" width="450px") (height e width é a escala da imagem)

  • [a] adiciona um link Exemplo: (a href="https://github.com/">Gitibas)

  • [h1, h2, h3...h6] para criar títulos em páginas web. O significado dessas tags é o mesmo, que é definir títulos,
    mas o nível de importância é diferente para cada uma, onde h1 tem o nível mais alto e h6 o mais baixo.
